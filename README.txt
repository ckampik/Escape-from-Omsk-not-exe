Данный репозиторий представляет из себя исходный проект на Unreal Engine 5.4 - игру Escape from Omsk,
в который рассказывается история про некого Сашу, цель которого сбежать из города Омска, вам предстоит пройти немало испытаний для прохождения игры.

Для того чтобы окунуться в мир Escape from Omsk следуйте инструкции ниже:

1. Для установки требуется скачать архив с Google disk по ссылке https://drive.google.com/file/d/1_3rRJqyXAlbeJeT_QMGSLqSM1SuTzP6l/view?usp=sharing.
2. Распаковать архив в нужную папку.
3. Запустить файл "Escape_from_Omsk.exe".

Перед запуском программы требуется установить пакеты visual studio (является необязательным пунктом, т.к. будут установлены автоматически при запуске игры).
Если возникли трудности при запуске или вы столкнулись с ошибкой в игре - обращйтесь в телеграм @zxcwed или @SkillNotFound.
